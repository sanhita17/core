Directive: Values: Usage: Include if wanting to...
show-all-strands true / false ... show a selectable item for "All Strands"
show-strand-as-breadcrumb true / false ... make the selector a breadcrumb to return to a previous or different view (or not)
hide-strand-drop-down true / false ... make the selector button unavailable
If directives are present, but unset (not true or false) the default value shown in GREEN will be used.
